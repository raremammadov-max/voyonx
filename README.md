**Voyonx**

Voyonx is a web-based travel platform focused on discovering places across Azerbaijan.
It provides an interactive 3D map experience with categorized locations, routing, and user-based features.


**Features**
Interactive 3D Map (Mapbox GL)
Location categories
Search functionality
Route building
Favorites system
Visited places tracking
User authentication
Responsive design (mobile-ready)


**Tech Stack**

_Frontend_
React
TypeScript
Tailwind CSS

_Backend_
Supabase
PostgreSQL

_Other Tools_
Mapbox GL
Vercel (Deployment)


**Live Version**
https://voyonx.com


**Environment Variables**
To run this project locally, create a .env file and add:

VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_MAPBOX_TOKEN=
